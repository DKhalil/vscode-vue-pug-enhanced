## Features

- Code snippets for .vue files with pug template.
- Transform html to pug or pug to html.

## Supported languages

- Pug
- Jade
- Vue

## Usage

### **Format**

1. Format selection.
2. Transform html to pug or pug to html

![](./doc/images/001.gif)

**command**:

- Html to Pug
- Pug to Html
